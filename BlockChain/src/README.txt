Objective: This program uses the contents of a text file to create a block chain.

----------------------------------------------------------------------

The following files are required in order to run the program:
	1) Block.java
	2) MainDriver.java
	3) StringUtil.java
	4) BlockChainData.txt

----------------------------------------------------------------------

How to run the program:
	1) Download all of the files from GitHub.
	2) Create a new Java project on Eclipse.
	3) Place the files into the src folder.
	4) Compile and run the MainDriver.java.

----------------------------------------------------------------------	
	
NOTE: This program is designed to read a text file with a specific format.

	Date: 03/08/2018
	Time: 00:00
	Moisture Level: 70%
	Water Level: 80%
	Temperature: 70 F 
	---			<-- This signifies the end of the block
